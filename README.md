# dio-excel-imposto-de-renda
Organizador de Imposto de Renda em Excel, desenvolvido como parte de um desafio da DIO. Este repositório contém a planilha com abas organizadas por tipo de rendimento e despesas, para facilitar a reunião de informações antes da declaração do IRPF.
Ideal para pessoas físicas que querem centralizar todos os informes em um só lugar, com iterface simples e navegação por menus.
